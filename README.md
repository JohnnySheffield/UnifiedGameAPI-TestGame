#Fake Game#

An awesome fake roulette game built with canvas to test the GameAPI integration.
Feel free to use!

##Installation##

    git@github.com:SPIL-stephanep/FakeGame.git
    cd FakeGame/
    npm install && bower install

##Run the game##

    npm start

Then point your web browser to: http://127.0.0.1:1337/
